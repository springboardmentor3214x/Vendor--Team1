# Vendor Reliability Intelligence Platform

## 📌 Objective
A full-stack web application to evaluate vendor reliability, manage procurement operations, monitor supplier performance, and improve procurement decision-making.

## 🛠️ Tech Stack
- Frontend: Angular
- Backend: FastAPI (Python)
- Database: PostgreSQL
- Version Control: Git & GitHub
